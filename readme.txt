This plugin is paid membership pro extension for payment gateways.

== Installation ==
Installation

Login your Wordpress admin and navigate to Plugins->Add New 

Find the Paytm payment for Paid Membership Pro plugin and activate the module.

After the successful activation of module, navigate to "Memberships->Payment Settings->Payment Gateway select Paytm Pay .

For Create Merchant ID Go to <a href="https://business.paytm.com/payments" target="_blank">Paytm Website</a>.

* Signup/Sign In on paywithpaytm.com and fill in the profile details and bank details.
